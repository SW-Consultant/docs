Политика конфиденциальности

Дата редакции: 26.10.2025

Настоящая Политика конфиденциальности определяет порядок сбора, хранения, обработки и использования персональных данных пользователей Telegram-бота SW Consultant (далее — Бот), разработанного Класс Глеб Станиславович (далее — Разработчик).

1. Общие положения

1.1. Используя Бота, Пользователь выражает согласие на обработку своих персональных данных в соответствии с настоящей Политикой.

1.2. Разработчик соблюдает конфиденциальность получаемой информации и принимает все разумные меры для её защиты.

2. Состав обрабатываемых данных

2.1. Бот обрабатывает следующие данные:

  • Telegram ID пользователя (автоматически);
  • имя, никнейм и иные сведения, предоставленные пользователем добровольно;
  • ответы на вопросы, тексты, запросы и технические данные, необходимые для работы Бота (время обращения, язык интерфейса).

2.2. Бот не собирает и не обрабатывает пароли, банковские реквизиты или любую информацию, не относящуюся к использованию Бота.

3. Цели обработки данных

Данные обрабатываются исключительно для:

  • предоставления функций и персонализированных ответов Бота;
  • анализа анонимной статистики для улучшения качества работы сервиса;
  • технической поддержки и связи с пользователями при необходимости.

4. Хранение и защита данных

4.1. Данные пользователей хранятся на защищённых серверах, доступ к которым имеет только Разработчик.

4.2. Срок хранения данных — до момента удаления пользователем или прекращения использования Бота.

4.3. Пользователь может в любой момент запросить удаление своих данных через контактные данные Разработчика.

5. Передача данных третьим лицам

5.1. Разработчик не передаёт персональные данные третьим лицам, за исключением случаев, прямо предусмотренных законодательством.

5.2. Любая передача данных по инициативе пользователя осуществляется только с согласия самого пользователя.

6. Заключительные положения

6.1. Разработчик оставляет за собой право обновлять Политику конфиденциальности.

6.2. Продолжение использования Бота после внесения изменений означает согласие с новой редакцией Политики.
